# CNS controlled-substance scheduling — the patterns that shape a program

DEA scheduling reshapes the trial path, manufacturing, prescribing, and commercial model long before approval, so the `dea-scheduling` agent needs the *current schedule + the documented action that set it*. The CNS-relevant landscape resolves into a few repeating patterns. All facts below were read from the **fetched `.gov` primary** (GPO `govinfo.gov` HTML — Federal Register rules + codified CFR), retrieved 2026-06-25; each quote is verbatim from the cited document.

## Pattern 1 — Schedule I psychedelics/hallucinogens (the hardest research constraint)
Codified at **21 CFR 1308.11(d)** (Schedule I hallucinogenic substances):
- **Psilocybin** — Schedule I, DEA code 7437 ([CFR 1308.11(d)](https://www.govinfo.gov/content/pkg/CFR-1996-title21-vol9/html/CFR-1996-title21-vol9-part1308.htm)). FDA Breakthrough-designated research + state programs (Oregon) do **not** change the federal listing.
- **LSD** — Schedule I, code 7315 (same CFR).
- **MDMA** — Schedule I, code 7405 (same CFR); DEA reaffirmed Schedule I by 1988 final rule after *Grinspoon v. DEA*. Context (not from the CFR primary): FDA issued a **2024 Complete Response Letter** to Lykos declining MDMA-assisted PTSD therapy — so MDMA stays Schedule I with no approved product.

**Impact:** Schedule I = the most restrictive research/handling controls (Schedule I registration, quota, secure storage) — a hurdle to surface, never a veto.

## Pattern 2 — the FDA-approval → scheduling pathway (a CNS drug gets a *less*-restrictive schedule)
When FDA approves a product whose molecule is otherwise tightly controlled, DEA places the **approved product** in a lower schedule:
- **Cannabidiol / Epidiolex** → **Schedule V** (Sept 2018, [83 FR 48950](https://www.govinfo.gov/content/pkg/FR-2018-09-28/html/2018-21121.htm)) — later effectively descheduled after the 2018 Farm Bill's hemp redefinition (verify current status before relying on Schedule V).
- **Dronabinol / Marinol** → **Schedule III** capsules (July 1999, [64 FR 35928](https://www.govinfo.gov/content/pkg/FR-1999-07-02/html/99-16833.htm)); note formulation-specificity — the 2017 Syndros *oral solution* sits in Schedule II.
- **FDA-approved marijuana products** → **Schedule III** (effective **April 28, 2026**, [the 2026 DEA/DOJ final order](https://www.govinfo.gov/content/pkg/FR-2026-04-28/html/2026-08176.htm)); botanical marijuana itself stays Schedule I unless FDA-approved.

**Impact:** the schedule attaches to the *approved product*, and can be **formulation-specific** — the same molecule may sit in different schedules across products.

## Pattern 3 — the split schedule (one molecule, two schedules)
- **GHB / sodium oxybate (Xyrem)** — GHB is **Schedule I** in bulk, but an FDA-approved GHB product is **Schedule III** *with Schedule I criminal sanctions* ([65 FR 13235](https://www.govinfo.gov/content/pkg/FR-2000-03-13/html/00-5925.htm), Mar 2000, per Pub. L. 106-172). Marketed under a restricted-distribution REMS.

## Pattern 4 — control by isomer-reference (no drug-specific action needed)
- **Esketamine (Spravato)** — the (S)-isomer of ketamine; controlled **Schedule III by reference** under the 1999 ketamine rule ([64 FR 37673](https://www.govinfo.gov/content/pkg/FR-1999-07-13/html/99-17803.htm), which controls ketamine "including its salts, isomers, and salts of isomers"). No standalone esketamine scheduling action exists.

> **Scope note.** This corpus covers the stable scheduling record for the major CNS-relevant controlled compounds; in-flight rescheduling petitions, analog-act ambiguity, and the biomedical literature behind abuse-potential findings are live-search / EMET gaps (see `manifest.md`).
