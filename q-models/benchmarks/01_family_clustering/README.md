## The Quiver question
Given a gene panel (40-gene CRISPR-N; eventually 1,400), which embeddings put same-family genes together?

## Metric
Leave-one-out nearest-neighbour same-family recall on the 40-gene CRISPR-N panel. Embeddings are mean-pooled + cosine; **best layer reported** (last-layer mean-pool undersells by ~0.10).

## Lesson / architectural rule
**Layer selection, not model scale, is the lever.** 650M ≈ 3B ≈ 15B (~0.85-0.875); the panel is saturated by its design (2 singleton families + functional-not-fold e3/NR groups). ESM-3's function track is the only thing to crack NRs (1.0) — the frontier is function-aware models, not bigger sequence models.
