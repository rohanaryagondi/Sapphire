This step predicts acute CNS toxicity scores for ASO candidates using a Gradient Boosting Regressor (GBR) model trained on Ionis patent datasets (UBE3A ATS and GFAP ASOs, n=894 total). Two scores are reported per sequence: the empirical Hagedorn score and the Zhang GBR score.

The model and notebook are located in the tox_predictions directory:
	ASO_tox_prediction_pipeline.ipynb   - Jupyter notebook (run this)
	aso_tox_gbr_model.pkl               - pre-trained GBR model (auto-loaded by notebook)

Output is saved to:
	tox_predictions/target_tox_predictions.csv


SCORES

Two toxicity scores are computed for each ASO sequence:

Hagedorn_predict_toxscore
	Empirical formula developed by Hagedorn et al., based on nucleotide composition:
		score = 136.0430 - 3.1263*(count A) - 5.1100*(count C) - 4.7217*(count T)
		                 - 10.1264*(count G) + 1.3577*(G-free 3' length)
	Higher score = lower predicted toxicity. No fixed threshold; used as a continuous reference.

GBR_predict_toxscore
	Zhang GBR model score. Trained on acute tolerance scores (0-7 scale) from Ionis patent ASOs.
	Uses 10 sequence features: MaxLength_A/T/G/C, Number_A/C/T/G, Length_Gfree_5, Length_Gfree_3.
	Threshold: score > 1.0 = Toxic (Ionis scale). Alternative threshold: score > 4.0 (Roche scale).
	Lower score = lower predicted toxicity.

The most important predictors in the GBR model are Number_G (~40% importance) and Length_Gfree_3
(~26% importance), consistent with known G-quadruplex-mediated CNS toxicity mechanisms.


INPUT

The notebook accepts a CSV file with a column named Sequence (or sequence, or ASO).
Example input format:

	Sequence
	GGTGAATCTTTATTAAAC
	TTGCTCCACCTTGGCCTGGCA
	GCGCAGGTAATCCCAAAAGCG

A sample input file can be generated by running the Appendix cell in the notebook.


RUNNING THE NOTEBOOK

The notebook requires Python 3 with the following packages: pandas, numpy, scikit-learn, joblib,
matplotlib, seaborn. These are standard in most Anaconda/JupyterLab environments.

Place the notebook and pkl file in the same directory, then:

1. Open the notebook in JupyterLab or Jupyter Notebook.
		jupyter lab ASO_tox_prediction_pipeline.ipynb

2. Set the input file path in Step 1 (the only line that needs to change between runs).
		INPUT_CSV = "your_sequences.csv"

3. Run all cells.
		Kernel > Restart & Run All

The notebook will automatically detect aso_tox_gbr_model.pkl in the same directory and load it,
skipping model training. If the pkl is not found, the notebook will retrain the GBR model from
scratch using the two Ionis training datasets (UBE3A_ATS_ASO_info_v2.csv and
GFAP_ASO_withICV_score_v2.csv, which must also be present in the same directory), save a new pkl,
and proceed to prediction. Subsequent runs will load the saved pkl automatically.


OUTPUT

Results are saved to tox_predictions/target_tox_predictions.csv (directory is created automatically
if it does not exist). The file contains at minimum the following three columns:

	Sequence                    - ASO sequence (uppercase DNA)
	Hagedorn_predict_toxscore   - Hagedorn empirical score (continuous; higher = less toxic)
	GBR_predict_toxscore        - Zhang GBR predicted acute tolerance score (lower = less toxic)

Additional columns included in the output: Tox_label (Toxic/Non-toxic based on GBR threshold),
and all 10 extracted sequence features (MaxLength_A/T/G/C, Number_A/C/T/G, Length_Gfree_5,
Length_Gfree_3). Sequences are sorted by GBR_predict_toxscore ascending (least toxic first).

Example output rows:

	Rank  Sequence                  Hagedorn_predict_toxscore  GBR_predict_toxscore  Tox_label
	1     GGTGAATCTTTATTAAACTA      57.5093                    0.346                 Non-toxic
	2     GGTGAATCTTTATTAAAC        62.6419                    0.563                 Non-toxic
	3     AGCGCAGGTAATCCCAAAAGC     21.6549                    2.383                 Toxic
	4     TTGCTCCACCTTGGCCTGGCA     12.6636                    2.821                 Toxic
	5     GCGCAGGTAATCCCAAAAGCG     13.2971                    4.338                 Toxic


MODEL DETAILS

Training data:   UBE3A ATS ASOs (n=555) + GFAP ASOs (n=339), from Ionis patents
Algorithm:       GradientBoostingRegressor (scikit-learn), hyperparameters selected by GridSearchCV
Best params:     n_estimators=90, max_depth=3, learning_rate=0.1, random_state=0
Training shuffle: random_state=42
CV R² (5-fold):  ~0.55
Reference:       Zhang et al., Quiver Bioscience (2024)
